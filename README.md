# sush-s-galleria

This repository contains my personal portfolio website, built to showcase all the web projects I have completed so far.  
Each project includes a short description, technology stack, and links to live demos or repositories.

---

## Features

- Clean, responsive portfolio website  
- Sections for project details, tech stack, and links  
- Fully customizable design  
- Simple structure for adding new projects  
- Fast and lightweight frontend

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript (or React, if applicable)
- **Hosting:** GitHub Pages / Vercel / Netlify
- **Assets:** Images, mockups, and UI previews for each project
